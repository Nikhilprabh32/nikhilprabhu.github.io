SwipeChef

Meal Prep on Steroids

Overview

SwipeChef is a mobile app that transforms meal prep into a fast, fun, and automated experience. Users swipe through recipe options, get personalized meal plans, and auto-generate grocery lists.

My Role

I worked on building the recipe recommendation engine using machine learning and designing a clean, swipe-based UI.

Tech Stack

  • React Native
  • Firebase
  • TensorFlow Lite (recommendation model)

Key Features

  • Swipe-based recipe discovery
  • Personalized meal planning
  • Automatic grocery list generation
